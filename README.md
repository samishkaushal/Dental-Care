# Dental Care Landing Page

## Overview

A modern, responsive landing page built for dental clinic owners to generate leads and increase patient bookings.

## Features

* Responsive Design (Mobile & Desktop)
* Hero Section with CTA
* Services Section
* Benefits Section
* Lead Generation Form
* How It Works Section
* Testimonials
* WhatsApp CTA
* Clean HTML, CSS and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Objective

The goal of this project is to create a high-converting landing page that helps dental clinics attract more patients and generate qualified leads through a Free Growth Audit offer.
